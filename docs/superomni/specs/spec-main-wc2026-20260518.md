# Spec: WorldCup 2026 AI Skills System

**Branch:** main | **Session:** wc2026 | **Date:** 2026-05-18

## Goal

Build a cross-platform, installable AI football intelligence skill system (worldcup2026-skills) for the 2026 FIFA World Cup.

## Requirements

### Execution Modes
This project is an installable AI Skill supporting the following execution modes:
1. **npx CLI** — `npx worldcup2026-skills predict "Brazil vs Argentina"`
2. **Claude Skills** — load `CLAUDE.md` in Claude Projects
3. **GitHub Copilot** — loaded via `.github/copilot-instructions.md`
4. **Other AI platforms** — native model support, no adapter required

### Core Capabilities (6 Skills)
1. **team-analysis** — national team strength modeling and tactical analysis
2. **schedule-fetcher** — fixtures / standings / knockout bracket
3. **player-analysis** — player rating, impact score, injury risk
4. **match-prediction** — multi-model fusion prediction engine
5. **analytics-engine** — multi-agent orchestration core brain
6. **data-provider** — data source abstraction layer

### Prediction Models (5 Models)
- ELO dynamic rating
- Bayesian probability model
- Expected goals (xG) model
- Monte Carlo simulation (≥10,000 iterations)
- ML ensemble (gradient boosting + logistic regression + neural probability)

### Self-Evolving System (5 Layers)
- Layer 1: Prediction engine (multi-model)
- Layer 2: Real result collector
- Layer 3: Prediction performance evaluator
- Layer 4: Self-learning optimizer (reinforcement learning + Bayesian update)
- Layer 5: Evolution memory system

### Multi-Agent Architecture (8 Agents)
Orchestrator → DataScout → TeamIntelligence + PlayerAnalyst → PredictionScientist → SimulationAgent → DebateAgent → MemoryAgent

## Tech Stack
- Node.js (CommonJS, npx-compatible)
- Zero external runtime dependencies (pure algorithm implementation)
- JSON Schema structured output
- Platform-agnostic Skill design
