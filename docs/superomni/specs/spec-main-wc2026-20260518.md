# Spec: WorldCup 2026 AI Skills System

**Branch:** main | **Session:** wc2026 | **Date:** 2026-05-18

## Goal

构建一个跨平台、可安装的 AI 足球智能技能系统（worldcup2026-skills），服务于 2026 FIFA 世界杯。

## Requirements

### 执行方式
本项目是一个可安装的 AI Skill，支持以下执行模式：
1. **npx CLI** — `npx worldcup2026-skills predict "Brazil vs Argentina"`
2. **Claude Skills** — 在 Claude Projects 中引入 `CLAUDE.md`
3. **GitHub Copilot** — 通过 `.github/copilot-instructions.md` 加载
4. **其他 AI 平台** — 该平台原生支持该模型，无需适配器

### 核心能力（6 Skills）
1. **team-analysis** — 国家队战力建模、战术分析
2. **schedule-fetcher** — 赛程/积分榜/淘汰赛支线
3. **player-analysis** — 球员评分、影响力、伤病风险
4. **match-prediction** — 多模型融合预测引擎
5. **analytics-engine** — 多智能体编排核心大脑
6. **data-provider** — 数据源适配抽象层

### 预测模型（5 Models）
- ELO 动态评分
- 贝叶斯概率模型
- 预期进球 xG 模型
- 蒙特卡洛模拟（≥10,000 次）
- 机器学习集成（梯度提升 + 逻辑回归 + 神经概率）

### 自进化系统（5 Layers）
- Layer 1: 预测引擎（多模型）
- Layer 2: 真实结果收集器
- Layer 3: 预测性能评估器
- Layer 4: 自学习优化器（强化学习 + 贝叶斯更新）
- Layer 5: 进化记忆系统

### 多智能体架构（8 Agents）
Orchestrator → DataScout → TeamIntelligence + PlayerAnalyst → PredictionScientist → SimulationAgent → DebateAgent → MemoryAgent

## Tech Stack
- Node.js (CommonJS, 兼容 npx)
- 零外部运行时依赖（纯算法实现）
- JSON Schema 结构化输出
- 平台无关的 Skill 设计
